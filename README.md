## Otsustusprotsessi toetava rakenduse loomine

See rakendus on loodud Tallinna Ülikooli bakalaureusetöö raames. Rakenduse eesmärk on toetada kasutajaid keeruliste otsuste tegemisel, kasutades **Analüütilise Hierarhia Protsessi (AHP)** meetodit.

## Projekti ülevaade

Rakendus võimaldab kasutajatel struktureerida keerulisi valikuprotsesse, jagades need kriteeriumideks ja alternatiivideks. Kasutades paarikaupa võrdlusi ja matemaatilist maatriksanalüüsi, arvutab süsteem välja kõige optimaalsema lahenduse vastavalt kasutaja eelistustele.

### Põhifunktsioonid:
- **AHP meetodi rakendamine:** Matemaatiliselt täpsed paarikaupa võrdlused Saaty skaalal (1–9).
- **Kasutajate autentimine:** Pilvepõhine sisselogimine ja registreerimine (Supabase integreerimine).
- **Otsuste salvestamine:** Võimalus salvestada oma otsustusprotsesse ja neid hiljem uuesti vaadata.
- **Mitmekeelsus:** Rakendus on täielikult lokaliseeritud nii eesti kui ka inglise keelde.
- **Ristplatvormilisus:** Loodud .NET MAUI raamistikul, toetades nii Androidi kui ka teisi platvorme.

## Tehnoloogiad

- **Raamistik:** .NET MAUI 
- **Keel:** C# / XAML
- **Andmebaas ja API:** Supabase
